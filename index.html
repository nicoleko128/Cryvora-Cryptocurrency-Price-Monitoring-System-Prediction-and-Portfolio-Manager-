CRYVORA

CRYVORA is a cryptocurrency analytics platform that provides real-time market monitoring, technical analysis, price prediction, and portfolio management tools. Designed to simplify complex market data, the platform helps users track performance, evaluate trends, and make informed investment decisions.

Real-Time Analytics Portfolio Prediction
About Us

Cryvora is an academic project demonstrating real-time financial data visualisation, technical analysis, and predictive modelling using publicly available APIs.

Built with Bootstrap 5, ApexCharts, and vanilla JavaScript — no server-side dependencies required.

Data Sources
  • Coinlore API Market data, tickers & global stats
  • Binance Public API OHLCV candlestick & kline data
  • Alternative.me Crypto Fear & Greed Index
  • CoinCap CDN Cryptocurrency logo images
  • Bootstrap 5.3 Responsive UI framework
  • TradingView Lightweight Charts Main price, volume, RSI & MACD charts
  • ApexCharts 3.44 Prediction, portfolio & comparison charts
